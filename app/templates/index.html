Simple Lists

Lightweight shared lists for groceries, packing, chores — no accounts, just links.

How it works

  1. The host logs in with the password set when the container starts.
  2. Create lists and copy a share link for family or friends.
  3. Everyone can add items and check them off. Optional per-list passwords available.
  4. Export and import all lists as JSON for backup or migration.
